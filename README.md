# Amazon Frontend Clone

This project is a simple frontend clone of the Amazon Homepage created using **HTML** and **CSS**.  
The aim of the project is to practice responsive layout, UI design, and web page structuring.

## 🚀 Features
- Fully responsive homepage layout
- Header, search bar, product grid, and footer design
- Login & account section UI
- Product card hover effects
